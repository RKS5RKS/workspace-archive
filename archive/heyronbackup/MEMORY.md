# System Capabilities & Limitations

## Image Generation
- **gpt-image-1-mini IS configured** in openclaw.json under models.providers.openai
- BUT the image tool only works for ANALYZING images, not GENERATING from text prompts
- Requires an actual image file to be uploaded to work
- **No text-to-image (DALL-E style) generator available** in this toolkit
- If Ryan wants images generated, he must use external tools (DALL-E, Midjourney, Canva AI) and send me the result to analyze/refine

## Browser
- Browserless profile configured but may not reach
- Chrome extension method works for Ryan to control

## What Works
- Web search
- Content writing
- Code (but limited libraries)
- Sub-agents
- File management
- GitHub integration

## What Doesn't Work
- YouTube video access (blocked)
- Image generation from text
- Installing new system packages
