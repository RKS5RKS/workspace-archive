# Pricing & Proposals

## 1. Pricing Calculator Prompts

### Value-Based Pricing Formula
```
Create a value-based pricing framework for a [service type] project.

Include:
- How to calculate ROI for the client (with examples)
- Questions to ask to quantify their potential gains
- Price anchoring techniques
- Three-tier pricing structure with justification for each tier
- How to present prices without bargaining

Make it so a freelancer can plug in numbers and get a price
```

### Hourly Rate Calculator
```
Create an hourly rate calculator for [freelance niche].

Inputs needed:
- Monthly income goal
- Billable hours per week (realistic - not all hours are billable)
- Business expenses (software, insurance, etc.)
- Taxes (estimate)
- Vacation/sick time buffer

Output:
- Minimum hourly rate
- Comfortable hourly rate
- Premium rate (for specialized work)

Include: how to present rates professionally
```

## 2. Tiered Pricing Builders

### Service Tier Builder
```
Create a 3-tier pricing structure for [service type].

For each tier provide:
- Tier name
- Price point
- What's included
- Best for (client type)
- Delivery time
- Revision rounds
- Add-ons available

Make tiers logical: basic/standard/premium with clear differentiation
```

### Retainer Model Builder
```
Create a monthly retainer pricing model for [service type].

Include:
- Hours included per month
- Roll-over policy (yes/no, how much)
- Priority status
- Communication channels
- Reporting/updates included
- Additional hours rate
- Minimum commitment period
- Early termination terms

Compare: 10hr/mo vs 20hr/mo vs unlimited
```

## 3. Proposal Sections

### ROI Justification Section
```
Write a section for proposals that justifies pricing using ROI language.

For each point include:
- How to frame the investment
- Questions to ask client to identify their budget
- What to say when asked for a discount
- How to handle "too expensive" objections
- Proof points to cite

Tone: confident, value-focused, not defensive
```

### Timeline & Milestones
```
Create a project timeline template for [service type].

Include phases:
- Discovery/Onboarding
- First draft/revision 1
- Feedback incorporation
- Final revisions
- Launch/delivery

For each:
- Duration
- What's delivered
- What I need from client
- Approval gate

Add: what happens if client misses deadlines
```
