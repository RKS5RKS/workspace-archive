# Trending Niche Finders

## 1. Underserved Niche Finder
```
Analyze Redbubble/Teespring for underserved niches in [category]. 

Find:
- Sub-niches with under 10,000 results
- Categories with high search volume but low competition
- Price gaps (what's missing at $25-35 range)
- Design style gaps (what's not being made)

Provide: specific niche names, estimated competition level, demand indicators
```

## 2. Seasonal Trend Predictor
```
Predict upcoming seasonal trends for POD products.

For [upcoming season/holiday]:
- What themes will be popular
- What colors will trend
- What phrases/quotes will resonate
- What design styles will emerge

Include: timing recommendations, early-mover opportunities
```

## 3. Audience Interest Mapper
```
Map out audience interests for [target demographic].

Include:
- Primary interests (top 5)
- Secondary interests (top 10)
- Pain points they relate to
- Humor styles they enjoy
- Values they connect with

Make it specific enough to generate 20+ design concepts
```

## 4. Competitor Gap Finder
```
Analyze the top 20 sellers in [niche] on [platform].

Find:
- Common design elements (what everyone's doing)
- Missing design approaches
- Price points that work
- What customers complain about in reviews
- Opportunities to do something different

Provide: 10 specific gap opportunities
```

## 5. Micro-Niche Explorer
```
Find micro-niches within [broad category].

For each micro-niche provide:
- Specific interest/topic
- Audience size estimate
- Competition level (low/medium/high)
- Design complexity needed
- Price potential

Focus on niches with 1,000-50,000 total products
```

# Design Concept Generators

## 6. Minimalist Text Designs
```
Generate 20 minimalist text-based design concepts for [niche].

Each include:
- The text/phrase (short, punchy)
- Typography style suggestions
- Single color or duotone palette
- Placement idea
- Why it resonates with the audience

Focus on: simplicity, readability, emotional connection
```

## 7. Quote Design Builder
```
Create 15 inspirational quote design concepts for [niche/audience].

For each:
- Quote text (original or reframe of common)
- Author attribution (if applicable)
- Design treatment (minimal, illustrated, photo overlay)
- Color mood
- Best product type (t-shirt, mug, poster)

Make quotes fresh - not the same "Dream Big" everyone else does
```

## 8. Pet Interest Designs
```
Generate 20 design concepts for pet-related POD products.

Include:
- Dog breed specific designs
- Cat breed specific designs
- Pet owner humor ("Dog Mom", "Cat Dad")
- Pet personality types
- Rainbow bridge/pet memorial
- Multi-pet household designs

Make them emotionally resonant - pet owners are passionate buyers
```

## 9. Hobby/Interest Mashups
```
Create 10 "miche" design concepts combining [interest A] + [interest B].

Example: "Reading" + "Cats" = "Reading Cat" design

For each mashup provide:
- Concept description
- Visual approach
- Target audience for each half
- Product recommendation (what sells best)

Make them clever, not forced
```

## 10. Vintage/Nostalgic Designs
```
Generate 15 vintage-style design concepts for [niche].

Include:
- Decade-specific aesthetics (50s, 70s, 90s)
- Retro color palettes
- Nostalgic reference points
- Vintage illustration styles
- Typography from the era

Make them feel authentic, not "I made this in Canva"
```

# Keyword & Listing Optimization

## 11. SEO Title Generator
```
Create 10 SEO-optimized titles for [product type] designs in [niche].

Each title:
- Under 80 characters
- Include primary keyword
- Include secondary keywords
- Be descriptive but not keyword-stuffed
- Sound natural (like a human wrote it)
```

## 12. Tag Research Framework
```
Build a tag strategy for [product] in [niche].

Include:
- Primary tags (5 most important)
- Secondary tags (10 relevant)
- Long-tail tags (10 specific)
- Seasonal tags (when applicable)
- Competitor tags (what's working for others)

Prioritize: high volume, medium competition
```

## 13. Description Generator
```
Write a compelling product description for [design] in [niche].

Include:
- Hook (first line - grab attention)
- Features (what, where to wear/use)
- Benefits (why they need it)
- Audience fit (who is this for)
- Call to action

Length: 150-200 words, scannable format
```
