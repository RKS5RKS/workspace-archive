# Content Creation Prompts Section

## 1. YouTube Templates

### Script Outline - Educational Video
```
Create a complete YouTube script template for an educational video about [topic]:
- Runtime: [X] minutes
- Target audience: [description]
- Goal: [teach/convert/entertain]

Include:
- Hook (first 30 seconds)
- Intro (1 minute)
- Main content outline (timestamps)
- Visual aids suggestions
- Mid-video CTA
- Outro/End card

Word count guide for each section
```

### Shorts Idea Generator
```
Generate 30 YouTube Shorts concepts for [niche]:
- How-to tutorials
- Quick tips
- Myth busts
- Behind the scenes
- Product reviews
- Q&A snippets

Each as single sentence hook + 2-3 bullet points on execution
```

### Thumbnail Hooks
```
Design 10 thumbnail concepts for [video topic]:
For each provide:
- Main image suggestion
- Text overlay (limited to 6 words)
- Color scheme
- Emotional expression to show
- CTR-boosting elements

Make them clickable without being clickbait
```

## 2. Blog Post Templates

### Ultimate Guide Structure
```
Create an ultimate guide outline for [topic]:
Target: [word count], Ranking for: [keywords]

Structure:
- Introduction (problem + solution promise)
- Table of contents
- Section 1: [foundational concept]
- Section 2: [core strategy]
- Section 3: [common mistakes to avoid]
- Section 4: [advanced tactics]
- Section 5: [case studies]
- Section 6: [tools/resources]
- Conclusion + CTA

Include estimated word counts and internal linking opportunities
```

### Listicle Template
```
Design a "X Ways to [Achieve Result]" blog post structure:
[X] = odd number (typically 7, 9, 11, or 21)

For each item provide:
- Headline (benefits-focused, not list-focused)
- Hook sentence
- Main advice (50-100 words)
- Example or proof point
- Visual direction

Include: intro, transitions between items, conclusion, CTA
```

## 3. Podcast Prompts

### Episode Framework
```
Create a podcast episode planning template:

Episode Title: [topic]
Target Length: [minutes]

Pre-Show:
- Topic introduction
- 3 key points to hit
- Guest briefing (if applicable)

Show Structure:
- Icebreaker/Hook (2 min)
- Main topic discussion (X min)
- Quick break/Tip (2 min)
- Deep dive (X min)
- Listener question segment (X min)
- Closing + CTA

Post-Show:
- Timestamp highlights
- Key quote for promotion
- Social clips to extract
```

### Guest Interview Prep
```
Create guest interview preparation for [guest]:
- Background research questions
- 3 "killer questions" that lead to unique insights
- Follow-up paths based on possible answers
- Soft pitch questions for offer/natural product mention
- Promotion questions for their content

Make questions conversational, not interrogative
```

## 4. Social Media Content

### Twitter/X Carousel
```
Design a 10-slide carousel for [topic]:
Slide 1: Hook/Question
Slide 2-7: Value points
Slide 8: Proof/Example
Slide 9: Summary
Slide 10: CTA

Each slide: text content + visual direction
Make it work as standalone screenshots too
```

### LinkedIn Article Outline
```
Create a LinkedIn article structure for [topic]:
Target: [read time]

Opening: [hook approach]
Story/Angle: [personal/professional/controversial]

Key sections with insights:
- [Section 1]
- [Section 2]
- [Section 3]
- [Section 4]

Close: [reflection + CTA to engage]
Include: 3-5 discussion questions to prompt comments
```
