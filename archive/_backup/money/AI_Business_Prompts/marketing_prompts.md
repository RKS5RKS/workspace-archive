# Marketing Prompts Section

## 1. Ad Copy Generators

### Facebook/Instagram Ad Hook
```
Create 10 different ad hook variations for a [business type] targeting [audience]. Include:
- Problem-aware hooks
- Solution-focused hooks  
- Social proof hooks
- Curiosity gaps
- Fear-based hooks (subtle)

For each, provide the headline, first line of body copy, and suggested visual direction.
```

### Google Ads Keyword Themes
```
Generate 50+ long-tail keyword themes for a [product/service] business. Organize by:
- Problem keywords
- Comparison keywords
- How-to keywords
- Best [category] keywords
- [Location] specific terms

Include estimated search volume indicators (high/medium/low) and competition level.
```

### Retargeting Ad Copy
```
Write a 5-ad retargeting sequence for people who visited [website] but didn't convert. 
Each ad should:
- Address a different objection
- Include specific value proposition
- Have clear CTA
- Be under 150 words

Ad 1: Reminder of value
Ad 2: Social proof/case study
Ad 3: Limited time offer
Ad 4: Risk reversal (guarantee)
Ad 5: Last chance/Urgency
```

## 2. Email Sequence Builders

### Welcome Sequence (5 Emails)
```
Create a 5-email welcome sequence for a [business type] that:
- Introduces the brand voice
- Delivers immediate value
- Establishes authority
- Soft sells the main offer
- Creates urgency for next step

For each email provide: Subject line, Preheader, Body copy, CTA
Tone: [friendly/professional/playful]
```

### Abandoned Cart Sequence
```
Write a 3-email abandoned cart sequence for [product]. 
Include: Email 1 (1 hour after), Email 2 (24 hours), Email 3 (72 hours)

Each email should:
- Reference specific product
- Address common objections
- Include social proof
- Have clear CTA
- Vary in tone from friendly to urgent
```

### Post-Purchase Upsell
```
Create a post-purchase email sequence (4 emails) for [product]. 
Goal: Convert customer to higher-tier product or repeat purchase.

Email 1: Thank you + delivery info
Email 2: How to get most out of [product]
Email 3: Customer success story + soft upsell
Email 4: Time-limited upgrade offer
```

## 4. Landing Page Prompts

### Hero Section
```
Write 5 different hero section variations for [product/service]:

For each provide:
- Headline (primary)
- Subheadline
- Hero paragraph (2-3 sentences)
- CTA button text
- Optional social proof snippet

Vary by angle: Features-focused, Benefits-focused, Social-proof focused, FOMO-focused, Authority-focused
```

### Features- Benefits Matrix
```
Create a features-to-benefits mapping for [product].
Include:
- Feature name
- Benefit (in customer language)
- Emotional payoff
- Proof point

Make it scannable and compelling
```

### FAQ Section
```
Generate 15 FAQ questions and answers for [product/service].
Organize by:
- Pricing/Payment
- Product/Service details
- Shipping/Delivery
- Returns/Guarantees
- Technical/Support

Write in conversational tone, not corporate
```
